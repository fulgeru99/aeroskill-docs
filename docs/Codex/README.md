# Aeroskill Club Web Platform Planning Pack

This folder contains the V1 planning documentation for the Aeroskill Club web platform.

Aeroskill Club is planned as a polished general aviation membership platform with a strong CRM foundation underneath. The platform includes a public website, member portal, digital member card, subscription/payment flow, benefits directory, and admin CRM for members, partners, sponsors, flight schools, aerodromes, associations, contracts, communications, and fleet records.
